
 == Open MoBu ==

 Plugins and scripts for Autodesk MotionBuilder.

Supported versions
- 2014, 2015, 2016, 2017, 2018

Plugins
* relation box - poseReader
* relation box - ray Intersect
* relation box - spring
* relation box - RBF Interpolation
* device - project Tango
* constraint - driven
* manager - view camera locker (only 2017, 2018)
* manager - auto interpolation patch (only 2017, 2018)
* manager - post processing (only 2017, 2018)
* manager - references (only 2017, 2018)
* shader - super lighting
* fbx extension references (only 2017, 2018)
* solver - auto compute normals

Scripts\Neill3d
- AssignUVSet
- GlobalSceneRetiming
- MakeCircle

Script Startup Tools
- additive animation tool
- post processing tool
- cleanup manager
- reference advanced tool
- restructuring
- stay on floor
